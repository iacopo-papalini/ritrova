{# Button-opens-popover wrapper around the existing subject_picker typeahead. Lets the subject-detail header show only a button ("Merge…" / "Move to…") instead of a wide inline text input, then reveal the picker below on click. Auto-closes on subject-selected (dispatched by subjectPicker.pick), outside-click, or Escape. Required context: button_label text on the trigger button (string) button_class Tailwind classes for the trigger (default: ghost) picker_onselect raw JS body (async fn body) for subjectPicker's onSelect callback. `target` is bound to the picked subject. Popover closes automatically once the body returns (via the subject-selected event bubbled by subjectPicker.pick); the body can noop-return early and the popover still closes. popover_width Tailwind width class for the popover panel (default w-72). The picker expands to fill it. align 'left' | 'right' — which edge of the trigger the popover aligns against (default 'right'). #} {% set _button_class = button_class|default('rounded-lg border border-gallery-border px-3 py-1.5 text-sm font-medium text-gallery-text transition-colors hover:bg-gallery-card cursor-pointer') %} {% set _popover_width = popover_width|default('w-72') %} {% set _align = align|default('right') %}
